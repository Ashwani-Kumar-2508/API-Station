# API Station – Simple API Explorer

API Station is a clean and simple React-based website that brings several useful public APIs together in one place. It is designed as a learning project to explore how data is fetched and displayed in modern web applications using React.

## 🚀 Features

- **Weather Finder**: Get real-time weather updates for any city using the Open-Meteo API.
- **Meal Finder**: Search for delicious recipes and food ideas from TheMealDB.
- **Movie Search**: Look up details about movies and TV shows via the TVMaze API.
- **Music Preview**: Listen to song previews and find artists using the iTunes API.
- **Book Library**: Browse a vast collection of books using the Google Books API.
- **News Explorer**: Stay updated with the latest headlines via the Hacker News API.

## 🛠️ Built With

- **React**: For building the user interface.
- **React Router**: For seamless navigation between pages.
- **Vanilla CSS**: Simple and clean styling with Flexbox and Grid.
- **Public APIs**: Six different open-source APIs integrated for real-time data.

## 📁 Project Structure

```text
src/
├── components/   # Reusable parts like Navbar and Footer
├── pages/        # Main pages (Home, About, Contact, and API pages)
├── services/     # Logic for fetching data from APIs
└── styles/       # Simple CSS files for the website look
```

## 📜 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Ashwani-Kumar-2508/API-Station.git
   ```
2. Enter the project folder:
   ```bash
   cd API-Station
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the website:
   ```bash
   npm run dev
   ```
5. Open your browser to the link shown in the terminal!
